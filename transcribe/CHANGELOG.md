# Changelog

## [0.1.0] - 2025-01-11

### Added
- Initial release
- Transcribe audio files to text using whisper.cpp
- Support for medium and large-turbo models
- Automatic audio format detection and conversion using ffmpeg
- Configuration management via `~/.config/cli-programs/transcribe.toml`
- Config subcommands: `config show` and `config set`
